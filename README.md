# 🎓 College Result Portal

A smart and interactive **College Result Analysis System** built using Python and Streamlit.  
This project allows students to check their results and provides powerful data insights for analysis.

---

## 🚀 Live Demo
👉 (https://collegeresultdashboard-kacnayhpre6an56dvbev6i.streamlit.app/)

---

## 📌 Features

- 🔎 Search student by Roll Number  
- 🏆 Automatic Rank Generation  
- 📊 Performance Classification (High / Average / Low)  
- ✅ Pass / Fail Status Detection  
- 📈 Interactive Data Visualization (Plotly Charts)  
- 📄 Download Result as PDF  
- 🔐 Admin Panel for uploading new result data  
- 🧑‍🎓 Full student dataset view  

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit 🌐  
- Pandas 📊  
- Plotly 📈  
- FPDF 📄
